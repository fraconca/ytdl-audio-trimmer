# Youtube Audio MP3 Downloader + Audio Trimmer

#### Requisitos
Node.JS

#### Módulos
path 
@distube/ytdl-core
fluent-ffmpeg


1. Instale as dependências
No terminal, execute:

npm install

2. Rode o arquivo index.js
